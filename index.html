Partie 3 : Js - les fonctions

Les fonctions permettent d'éviter le répétement de code...

Par exemple : plutôt que de devoir écrire Math.floor(Math.random()*100) pour obtenir un chiffre entre 0 et 100 exclus

Nous pouvons créer une fonction !
Il n'y a aucune différence entre une fonction normal et fonction flechée, c'est juste 2 manières d'écrire la même chose. 

    //fonction normal
    function getRandomInt(min, max){
        return Math.floor(Math.random() * (max - min)) + min;
    }

    //fonction flechée
    const getRandomInt = (min, max) => {
        return Math.floor(Math.random() * (max - min)) + min;
    }

    let nombreAleatoire = getRandomInt(1, 101);
    //donne un nombre aléatoire entre 1 et 100

Dans l'exemple ci-dessus, on crée un fonction nommée "getRandomInt" avec 2 paramètres :

  • "min" qui sera la valeur minimal de l'aléatoire
  • "max" qui sera la valeur maximal excluse par l'aléatoire.

Mise en place et utilisation d'une fonction

Les class - Intro

En réalité, les classes sont juste des fonctions spéciales. Ainsi, les classes sont définies de la même façon que les fonctions : par déclaration, ou par expression.

MDN web docs

tout ce que je peux rajouter c'est un genre de "moule"

Par exemple on peut créer une class voiture avec des propriétés marque, model, couleur...

De cette on pourra en faire découller une voiture de marque Audi, model A3 de couleur blanche

Mais aussi si on le souhaite une autre voiture de marque Toyota, model Yaris de couleur grise

Par exemple nous allons créer une class Voiture avec différent attribut et une méthode Avancer parce que oui, une voiture ça avance...
(méthode et fonction font exactement la même chose mais on appele celà méthode quand c'est dans un objet).

    class Voiture{
        constructor(marque, model, couleur){
            this.marque = marque;
            this.model = model;
            this.couleur = couleur;
        }
        avancer(){
            console.log("vrooum");
        }
    }

    //Création d'une voiture de marque Audi
    let audi = new Voiture("Audi", "A3", "Noir");

    //Création d'une voiture de marque Toyota
    let toyota = new Voiture("Toyota", "Yaris", "Gris");

    //et les 2 voitures que l'on a crée possède la méthode "avancer"
    audi.avancer();
    toyota.avancer();
